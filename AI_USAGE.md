# AI 도구 활용 (AI_USAGE)

## 1. 사용한 도구

- ChatGPT
- Cursor AI

<br/>

## 2. 주로 어디에 썼나

**ChatGPT**

- 코드 리뷰 및 디버깅
- 성능 최적화
- 설계 방향 및 예외 처리 검토
- 문서 초안 작성

**Cursor AI**

- 테스트 코드 초안 작성
- Skeleton UI, 모달 등 UI 기본 구조 작성

<br/>

## 3. AI 답을 그대로 쓰지 않고 검증·거부·수정한 사례

### rollback 테스트 방식

- AI는 테스트를 위해 rollback 전용 순수 함수를 새로 만들어 사용하는 방식을 제안했음
- 하지만 테스트를 위해 실제 서비스 로직을 변경하는 것은 적절하지 않다고 판단함
- 기존 구현 방식을 유지한 채, 현재 구조에 맞는 순수 함수를 직접 작성하여 테스트를 진행함

<br/>

## 4. 스스로 판단해서 결정한 부분

- React Query, Redux 등 상태 관리 라이브러리를 사용하지 않고 React 기본 상태 관리 기능으로 구현
- HTML5 Drag & Drop API를 사용하여 별도 라이브러리 없이 구현
- Cursor AI는 Skeleton UI, 모달 등 반복적인 UI의 기본 형태를 작성하는 용도로 활용하고, 구조와 로직은 직접 구현 및 수정함
- Optimistic Update와 rollback 구조를 현재 프로젝트에 맞게 직접 구성
- AI가 제안한 내용은 요구사항과 프로젝트 구조에 맞는지 검토한 뒤 필요한 부분만 반영
